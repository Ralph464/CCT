ceasar

To create this encrypt/decrypter in C, you need a great understanding in simple array and function consequences. This program takes a sequence of letters and an integer. The program shifts the sequence the number given. So if you where to input A sequence of letters such as ABC and the number 3; A, B and C values would be changed up 3 times. So A would be D, B would be E, and C would F. This function applies to both uppercase and lowercase letters, neglecting numbers.
